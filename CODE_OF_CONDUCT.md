# 社区行为准则

为构建友好、专业、可持续的开源协作环境，参与者需遵守以下规范。

## 我们倡导

- 尊重他人，聚焦问题本身
- 接受建设性反馈
- 理解不同经验背景与观点
- 以协作为目标推进讨论

## 不可接受的行为

- 人身攻击、侮辱、歧视、骚扰
- 恶意挑衅、引战、持续破坏讨论秩序
- 泄露他人隐私或敏感信息
- 任何违反适用法律法规的行为

## 适用范围

适用于本仓库的 Issue、PR、Discussion、评论区等所有协作场景。

## 处理方式

维护者有权对违规内容进行编辑、隐藏、删除，并视情况采取警告、临时限制或永久封禁。

## 联系方式

如遇不当行为，请联系维护者或发送邮件至：`maintainer@example.com`。
