# 前端自定义装修功能说明

## 1. 功能定位

前端自定义装修功能用于通过可视化方式配置商城首页/专题页布局，实现运营自主搭建页面，无需频繁发版。

## 2. 典型场景

- 首页活动主题改版
- 节日专题页配置
- 商品推荐位动态调整

## 3. 前端能力范围

- 页面模板选择（首页、会场页、落地页）
- 组件拖拽与排序（轮播、宫格、商品、公告、富文本等）
- 组件属性编辑（颜色、间距、数据源、跳转链接）
- 页面预览、保存草稿、发布上线

## 4. 建议页面结构

- 装修页面列表
- 装修编辑器
- 预览与发布面板
- 发布记录与回滚

## 5. 权限建议

建议将权限拆分如下：

- `mall:decorate:page:list`
- `mall:decorate:page:create`
- `mall:decorate:page:update`
- `mall:decorate:page:publish`
- `mall:decorate:page:rollback`

## 6. 数据模型建议

- 页面：页面标识、页面类型、状态、版本号、发布时间
- 组件：组件类型、排序、配置 JSON、数据源配置
- 发布记录：操作人、版本、发布时间、备注

## 7. 风险与注意事项

- 配置 JSON 需版本化，避免新旧组件字段不兼容
- 发布后建议支持快速回滚
- 线上配置应有白名单校验，避免非法跳转链接
